Notas.

El proyecto se conecta a una BD SQL y usa Dapper como ORM para la extracción de datos
La Base de datos se llama VisiotechDB
SE DEBE CAMBIAR EL STRING DE CONEXIÖN DE BD POR EL QUE SE VAYA A USAR
En el fichero appsettings.json podrás encontrar el string de conexión 
Tienes un fichero dockerfile donde se especifica donde se va a guardar la aplicación, en que puertos va a trabajar
Tiene la opción del Swagger para probar los distintos endpoint
Puedes ver los logs de la aplicación en el apartado de logs cuando ejecutas el contenedor
